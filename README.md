

## Description

This is the backend repo for the ResponAI messaging app. It is a NestJS repo, uses a psql database with typeorm, websockets, and is deployed through a docker container, serving https traffic through the caddy network docker image. 

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
